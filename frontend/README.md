# Frontend (not started — intentionally)

Per the execution rule, the frontend starts only after the backend golden
path is stable and tested (it is) **and** a sprint explicitly opens this
workstream (it hasn't).

Planned MVP (see `.claude/agents/frontend-agent.md` and
`docs/wiki/roadmap.md`): Next.js + React + Tailwind, React Three Fiber STL
viewer, prompt input, artifact list, Markdown report viewer, status panel —
all against the documented API (`docs/wiki/api.md`).
