# System Overview

**Source:** MECHANICAL ENGINEER AGENT — Technical Report
**Event:** AMD Developer Hackathon: ACT II | July 6–11, 2026
**Version:** 0.1 (Pre-build) | **Team:** 2 (AI/Agents Engineer + Mechatronics Engineer)

Mechanical Engineer Agent is a multi-agent AI system that converts a natural-language engineering request into a validated, manufacturable design package. The MVP targets one vertical demo flow:

```
Text prompt → Requirements JSON → System Architecture → Parametric CAD
→ Physics Simulation → Failure/Risk Report → BOM + Cost → STL Export
```

**Target demo prompt:** "Design a mobile robot that can inspect manufacturing equipment for 8 hours."
