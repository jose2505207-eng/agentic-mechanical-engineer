# Manufacturing Intelligence

## 3D Printability Analysis
Evaluate:
- Overhangs
- Support requirements
- Orientation
- Warping risks
- Material usage

Output: printability report

## CNC Manufacturability
Evaluate:
- Tool access
- Pocket depth
- Complexity

## Injection Moldability
Evaluate:
- Draft angles
- Wall thickness
- Undercuts

## Assembly Analysis
Evaluate:
- Serviceability
- Tool access
- Fastener accessibility
