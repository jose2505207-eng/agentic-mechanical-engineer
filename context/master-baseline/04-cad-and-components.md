# CAD Capabilities & Component Analysis

## CAD Capabilities
**Generate:** STEP, STL, OBJ, GLB

**Support:** assemblies, technical drawings, exploded views, manufacturing drawings

**Future:** Image to CAD, Text to CAD, CAD Repair

## Existing Component Analysis
**Accept:** STEP, STL, SolidWorks, Fusion files, AutoCAD exports

**Extract:** dimensions, interfaces, holes, mount points, constraints, assemblies

## Design Around Existing Components
**This is likely the strongest startup feature.**

Example — upload: Raspberry Pi, camera, battery.
Prompt: "Build a mobile inspection robot around these components."

Output:
- Chassis
- Mounting structures
- Cable management
- Assembly

This eliminates significant CAD work.
