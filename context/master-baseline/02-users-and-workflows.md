# Primary User Types & Core Workflows

## Primary User Types
1. Manufacturing Engineers
2. Mechanical Engineers
3. Robotics Engineers
4. Product Designers
5. Startup Founders
6. Research Labs
7. Students
8. Makers
9. Automation Engineers
10. Medical Device Engineers

## Workflow 1 — Text to Robot
**User:** "Design a robot that can carry 10 kg."
**Output:** robot architecture, motors, sensors, battery, CAD, simulation, BOM, manufacturing package

## Workflow 2 — Text to Product
**User:** "Build a camera inspection system."
**Output:** product concept, mechanical design, material recommendations, manufacturing package

## Workflow 3 — Existing Parts to Design
**User uploads:** camera, battery, servo, PCB
**Prompt:** "Build an inspection robot around these parts."
**Output:** mounts, chassis, assembly, enclosures, wiring plan

## Workflow 4 — CAD to Improvement
**User uploads:** STEP, STL, assembly
**Agent:** detects issues, finds weak points, suggests improvements, optimizes geometry

## Workflow 5 — CAD to Digital Twin
**User uploads a machine.**
**Agent generates:** physics model, kinematic model, dynamic model, simulation environment
